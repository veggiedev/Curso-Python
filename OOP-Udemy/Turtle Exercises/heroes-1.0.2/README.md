## Inspired by

- [Sindresorhus's Superheroes](https://github.com/sindresorhus/superheroes)

## Install

```
$ pip install heroes
```

## Usage

```python
import heroes

print (heroes.gen())
#=> Hooded

print (heroes.genarr(3))
#=> ['Askew-Tronics', 'Decepticon', 'Leopardon']
```
